## 🔄 HOW TO USE THIS CODEBASE

### 1. Clone the Repository
```bash
git clone https://github.com/antonymwangidev-hub/local-biz-platform.git
cd local-biz-platform-main
```

### 2. Install Dependencies
```bash
npm install
# or
bun install
```

### 3. Set Up Environment
```bash
# Create .env.local with:
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_PUBLIC_URL=http://localhost:5173
```

### 4. Apply Database Migrations
```bash
# In Supabase Dashboard, run migrations from supabase/migrations/
```

### 5. Start Development Server
```bash
npm run dev
```

### 6. Access the Application
```
http://localhost:5173
```

### 7. Use Demo Data
- Click "Seed Demo Data" button in admin panel
- 10+ sample businesses created
- 50+ posts, events, communities generated
